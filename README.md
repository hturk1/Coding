# School Projects Repository

This repository contains various projects and assignments I have worked on during my time at the University of Michigan–Dearborn. It serves as a collection of my school work, ranging from coursework exercises to larger projects.

## ABET_Portal Project

One of the main projects in this repository is the **ABET_Portal**, which was developed as part of a project for the University of Michigan–Dearborn. The ABET_Portal project focuses on streamlining the ABET accreditation process for faculty and staff at the University of Michigan-Dearborn. 

## Repository Structure

- `ABET_Portal/` – The main project directory for the ABET portal.
- Other folders – Coursework and school assignments.
